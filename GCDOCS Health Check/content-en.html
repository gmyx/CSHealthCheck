Health Check

Information for Health Check

This is the main page. It runs the Health Check on any GCDOCS instance. There are two functions:

  • Run Health Check on Selected Instances : This is the main function. Choose one or more instances to run the health check agaisnt.
  • Set-up Notifications on Selected Instances : This is only used once per instance. It will set-up notifications and e-mail address in your personal settings. It must be run at least once per instance.

Make sure you are running in a private browser window.

Password Change

Information for password changing

This page allows you to change your password on any GCDOCS instance. Since information is written to disk, you need to supply an enryption key that will be used in the process. This key will be asked again once inside of GCDOCS.

Information for settings
